# Coffee Shop Sales Analysis

## Dashboard

![Coffee Shop Dashboard](coffee_shop_dashboard.png)

## Overview
This project analyzes transactional sales data from a coffee shop to identify trends in product performance, revenue distribution, and sales patterns throughout the day.

The goal of this analysis is to demonstrate how data analysis and visualization can help businesses make better decisions about product offerings, promotions, and operations.

Dataset
The dataset contains simulated coffee shop transactions with the following fields:
Order ID
Date
Time
Location
Product Category
Product
Quantity
Price
Revenue
Day of Week
Month
Hour

Additional columns were created during the data cleaning process to support time-based analysis.

Data Cleaning
Data preparation steps included:
Standardizing time formats
Creating new fields such as Day of Week, Month, and Hour
Verifying revenue calculations
Structuring the dataset for pivot table analysis

Analysis
Using Excel pivot tables, the following analyses were performed:
Revenue by store location
Top-selling products by quantity
Revenue contribution by product category
Hourly sales trends

Dashboard
An Excel dashboard was created to visualize the results, including:
Total Revenue KPI
Revenue by Store Location
Top 10 Best-Selling Products
Revenue Contribution by Category
Hourly Sales Trend

Key Insights
The University location generated the highest revenue among all stores.
Sandwich products were among the top-performing items.
Coffee products contribute a significant share of total revenue.
Sales remain relatively consistent throughout the day with slight variations.
